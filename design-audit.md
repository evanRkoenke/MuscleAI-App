# Design Audit — Reference Image Analysis

## Dashboard (Left Phone in Reference)
- Header: "MUSCLE AI" in italic/bold serif-like font, settings gear icon right
- Calorie Ring: THICK ring (~12px stroke), blue gradient glow (dark blue to cyan), inner shadow/glow effect, large bold number "1,650" centered, "Calories Remaining" subtitle below
- Macro Cards: 3 cards in a row — "240g PROTEIN" (cyan), "232g CARBS" (amber/yellow), "72g FAT" (red/orange) — each with colored label below the number
- Quick Actions: 3 pill buttons in a row — "SCAN" (camera icon), "MEALS" (fork/knife icon), "FORECAST" (chart icon) — dark surface with subtle border
- Protein Priority Section: Card with food image (steak + broccoli), "ANABOLIC SCORE" badge showing "87" in green, overlay on image
- Bottom Tab Bar: 4 tabs — SCAN, MEALS, TRACK, FORECAST — with icons, active tab highlighted in blue
- Floating Camera Button: Large circular blue glowing button at bottom center, overlapping tab bar

## Forecast Screen (Right Phone in Reference)
- Header: "ANABOLIC FORECAST" with chart icon, lock icon for premium
- Weight Chart: Line graph showing weight progression from 180 lbs, data points with dots, x-axis "1M, 3M, 6M, 12 MONTHS", white/cyan line on dark background
- Priority Sync Card: Lock icon, "Locked content update" text
- Premium Upsell Box: "PREMIUM MEMBERS ONLY", "Unlock this 12-Month Forecast and multiply your gains.", "ELITE ANNUAL $79.99", "66% SAVINGS", large "UNLOCK" button with blue-to-red gradient

## Key Design Elements to Match
1. Calorie ring needs THICK stroke with blue gradient glow (not thin gray)
2. Macro cards need colored labels (cyan for protein, amber for carbs, red for fat)
3. Typography should be bold, condensed, high-tech feel
4. Blue glow effects on interactive elements
5. Floating camera button with glow
6. Dark surfaces with subtle borders, not flat
7. Electric Blue (#007AFF) as primary action color
